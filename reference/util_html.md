# Convert character string to html class

Convert character string to html class

## Usage

``` r
util_html(text)
```

## Arguments

- text:

  character string input

## Value

The same input character string with html class

## Details

Adapted from
[`HTML`](https://rstudio.github.io/htmltools/reference/HTML.html)

## Examples

``` r
util_html('abd')
#> abd
```
