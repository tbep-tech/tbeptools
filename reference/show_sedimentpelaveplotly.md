# Creates a plotly object for average PEL plots

Creates a plotly object for average PEL plots

## Usage

``` r
show_sedimentpelaveplotly(p, width = NULL, height = NULL)
```

## Arguments

- p:

  [`ggplot`](https://ggplot2.tidyverse.org/reference/ggplot.html) object
  as output from
  [`show_sedimentpelave`](https://tbep-tech.github.io/tbeptools/reference/show_sedimentpelave.md)

- width:

  numeric for width of the plot in pixels

- height:

  numeric for height of the plot in pixels

## Value

A [`plotly`](https://rdrr.io/pkg/plotly/man/plotly.html) data object

## Examples

p <- show_sedimentpelave(sedimentdata)
show_sedimentpelaveplotly(p)
